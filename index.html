My Canvas

Xierra's Links

Xierra's Links is a website designed with a concept similar to Linktree and themed around hackers. This project was developed without prior programming knowledge, only with the assistance of artificial intelligence (AI). CyberLinks connects users with various social media links on a single page, making it easy for users to direct visitors to their profiles and content on other platforms. With the success of this project, I have become increasingly interested in the world of web development.

Sulawesi Airlines

Sulawesi Airlines is a virtual platform crafted for members of a virtual airline (VA) community operating within an online flight simulation environment. This platform delivers an airline-style experience with various user roles, such as virtual pilots, ATC communicators, and aviation enthusiasts. Key features include registration, FLOPS, flight logs, and user profiles, allowing users to engage in an immersive, interactive aviation simulation.

Habil's Portfolio

This portfolio site serves as a creative showcase, highlighting a diverse array of my works and projects. Each project embodies a unique touch and originality, making this portfolio a dynamic reflection of my evolving style and artistry.
